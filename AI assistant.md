# AI Assistant with n8n + Ollama (Simple Guide)

This workflow connects `Ollama` with `n8n AI` to run a local AI assistant flow.

## What It Does

- Accepts input in n8n.
- Sends the prompt to Ollama.
- Returns an AI response in your workflow.
